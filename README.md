## Get Started

### Step 1. - Git clone ([Download git for Windows](https://git-scm.com/downloads))

    git clone https://github.com/micahh2/project-zelda.git

### Step 2. - Change directory

    cd project-zelda

### Step 3. Compile & Run

- with make: `make`
- with just javac:

        javac projectzelda/Main.java
        java projectzelda/Main.java
